# Assignment2SEG3125

Lumière Studio — a salon booking website built with React.

**Live site:** [https://murtazarhythm.github.io/Assignment2SEG3125/](https://murtazarhythm.github.io/Assignment2SEG3125/)

## Local development

```bash
npm install
npm start
```

Opens at [http://localhost:3000](http://localhost:3000).

## Deployment

Pushes to `main` automatically deploy to GitHub Pages via the workflow in `.github/workflows/deploy.yml`.

Ensure **Settings → Pages → Source** is set to **GitHub Actions**.
